# Accounting Service

A Go library providing accounting and bookkeeping functionality with support for multiple accounting service providers (SimplBooks, Merit).

## Installation

```bash
go get github.com/qbitsoftware/accounting-service
```

## Usage

```go
import "github.com/qbitsoftware/accounting-service"

// Create a client with your accounting service provider
client := accounting.NewClient(provider, apiKey)

// Use the service
// ... (add specific usage examples as needed)
```

## Features

- Customer management
- Invoice management
- Payment processing
- Purchase tracking
- Tax calculations
- Report generation
- Multi-provider support (SimplBooks, Merit)
